# 20 - Testing Strategy

## Phase 1: sts-ruby Round-trip Tests

### Test Harness

Location: `sts-ruby/spec/round_trip/`

```ruby
# sts-ruby/spec/round_trip/reference_docs_spec.rb
RSpec.describe "Reference document round-trips" do
  reference_docs = Dir.glob(File.join(reference_docs_path, "**", "*.xml"))

  reference_docs.each do |file|
    describe File.basename(file) do
      it "round-trips without data loss" do
        doc = File.read(file)
        parsed = Sts::IsoSts::Standard.from_xml(doc)
        generated = parsed.to_xml(
          pretty: true,
          declaration: true,
          encoding: "utf-8",
        )
        expect(generated).to be_xml_equivalent_to(doc)
      end
    end
  end
end
```

### Reference Document Fixtures

Copy all XML files from `mn-samples-iso-private/reference-docs/` into `sts-ruby/spec/fixtures/reference_docs/` preserving directory structure.

### Incremental Approach

1. Start with the simplest/smallest reference document
2. Fix one model gap at a time
3. Re-run all round-trips after each fix
4. Track which documents pass in a progress file

### Unit Test Pattern

Each transformer test follows the same pattern:

```ruby
RSpec.describe ParagraphTransformer do
  let(:context) { Transformer::Context.new(source_document) }

  it "transforms a simple paragraph" do
    source = create_paragraph(content: "Hello world")
    result = ParagraphTransformer.new(context).transform(source)

    expect(result).to be_a(Sts::IsoSts::Paragraph)
    expect(result.content).to include("Hello world")
  end

  it "transforms paragraph with inline formatting" do
    source = create_paragraph(content: [
      "Hello ",
      create_strong(content: "world"),
    ])
    result = ParagraphTransformer.new(context).transform(source)

    expect(result.bold).not_to be_nil
  end
end
```

### Integration Test Pattern

End-to-end tests use real Metanorma XML files and compare against reference STS XML:

```ruby
RSpec.describe "End-to-end transformation" do
  let(:input_xml) { File.read("spec/fixtures/iso_8601_1.input.xml") }
  let(:expected_xml) { File.read("spec/fixtures/iso_8601_1.expected.sts.xml") }

  it "transforms ISO 8601-1 to STS" do
    source = Metanorma::IsoDocument::Root.from_xml(input_xml)
    result = Metanorma::Iso::Sts::Transformer.transform(source)
    generated = result.to_xml(pretty: true, declaration: true, encoding: "utf-8")

    expect(generated).to be_xml_equivalent_to(expected_xml)
  end
end
```

### Test Fixtures for Transformer

Need paired input/output files:
- `spec/fixtures/iso_8601_1.input.xml` — Metanorma semantic XML
- `spec/fixtures/iso_8601_1.expected.sts.xml` — Reference STS XML (from mnconvert output)

These can be generated by running mnconvert on the sample documents.
