# 090: Missing stem rStyle for math formulas

## Problem
Reference uses `stem` rStyle (99 occurrences) for mathematical expressions. Output renders math as plain text without the `stem` character style.

## Reference:
```xml
<w:r>
  <w:rPr><w:rStyle w:val="stem"/></w:rPr>
  <w:t xml:space="preserve"> w </w:t>
</w:r>
```

## Output:
```xml
<w:r>
  <w:t>w</w:t>
</w:r>
```

No rStyle applied, and the content is plain text fallback from MathML.

## Root cause
1. `render_stem` in inline.rb falls back to plain text from the stem's text attribute
2. The `stem` rStyle is added to the template via `adapter_styles` but never applied
3. MathML content is not converted to OOXML math markup (`<m:oMath>`)

## Fix
1. Apply `stem` rStyle to runs generated by `render_stem`
2. For proper rendering, convert MathML to OOXML math (requires a MathML→OMML converter)
3. At minimum, wrap stem text runs with the `stem` rStyle

## Location
- `lib/isodoc/iso/docx/inline.rb` — `render_stem`
